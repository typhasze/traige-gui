# Python standard library is sufficient for this project
# The application uses only built-in modules:
# - tkinter (usually comes with Python)
# - os, sys, subprocess, json, threading, time
# - datetime, glob, shutil, signal, urllib.parse
# - typing (Python 3.7+)

# No external packages required!
# Note: tkinter should be installed via system package manager if not available:
# Ubuntu/Debian: sudo apt-get install python3-tk
# Fedora: sudo dnf install python3-tkinter
# macOS: included with Python from python.org
